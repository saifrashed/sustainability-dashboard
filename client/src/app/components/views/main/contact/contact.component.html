Frequently Asked Questions

How does the grading work?

As an admin you will have a graph for every faculty on a questionnare. Based on the answers the faculties will fill in you will get a result filled within graphs.

How does a faculty fill in the questionnares?

An admin can choose to set certain questionnaires available to faculties, allowing the faculty account owner have the questionnaire available to be filled in.

Can I edit the questionnaire?

Yes, you can edit the questionnaire. However if a single faculty already has filled in the questionnaire, you will no longer be able to edit the questionnaire.
